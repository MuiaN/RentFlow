Build a modern frontend-only web application called “RentFlow – Rental Management Dashboard (Demo)”.

This is a mock demo application for a rental management company. It must NOT connect to a real backend. Use mock data stored locally in JSON files or JavaScript objects.

🎯 PURPOSE OF THE APP

This is a demo dashboard for a rental management company admin and staff to:

Manage houses/units

Assign tenants to units

Automatically generate monthly invoices (mock logic)

Mark invoices as paid (cash or bank transfer)

Automatically generate receipts (mock)

Automatically show due and past-due invoices

Simulate automated notifications (UI simulation only)

The app must look realistic and production-ready but must use only mock data.
